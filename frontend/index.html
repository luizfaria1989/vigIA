Sobre o vigIA

Um instrumento de antecipação — não de detecção

O vigIA responde, todo dia, a uma pergunta só: nos próximos 5 dias, onde em Goiás é mais provável pegar fogo — e onde exatamente? O valor está no antes: tempo suficiente para deslocar uma brigada, emitir um alerta ou priorizar fiscalização.

Como ler o painel

Ranking → qual região
Os municípios com maior probabilidade no dia escolhido.
Mapa → onde dentro
Clique num município e veja as zonas internas com risco individual.
5 dias → quando
Troque o dia no topo: o mapa esquenta ou esfria, o ranking reordena.
Escala de fogo → quanto
Uma só escala — verde, âmbar, vermelho — no mapa, no ranking e na legenda.

Transparência sobre a incerteza

O risco é uma probabilidade — não uma certeza. O modelo tende a errar mais na estação chuvosa, quando o fogo é menos previsível. Olhe sempre a data de geração: este boletim é um instantâneo diário, atualizado uma vez por dia.

Dados e fontes

Focos de incêndio: BDqueimadas / INPE (histórico 2015–2025).
Dados climáticos: Open-Meteo (temperatura, precipitação, vento, umidade).
Malha municipal: IBGE (via geodata-br, domínio público CC0).
Imagens de satélite: Esri World Imagery (Maxar, Earthstar Geographics).
